# Создание заказа магазином

**Дата обновления:** 2025-01-26

## Порядок заполнения (по ТЗ заказчика)

1. **Адрес доставки** — магазин вводит адрес первым
2. **Цена** — автоматически по тарифу (геокод) или ручной ввод с валидацией
3. **Дата** — сегодня/завтра
4. **Имя получателя**
5. **Телефон получателя**
6. **Комментарий** — или /skip

## Валидация цены по тарифной сетке

- При ручном вводе: минимум = тариф для данного расстояния (из `app.tariffs.X` в application.properties)
- Если расстояние неизвестно (геокод не сработал) — минимум 300₽
- При вводе цены ниже минимума — бот показывает тарифную сетку и просит ввести корректную сумму

## ЮKassa (пополнение депозита курьера)

Уже подключена. Нужны только данные от заказчика:
- `yookassa.shop-id` — ID магазина из личного кабинета ЮKassa
- `yookassa.secret-key` — секретный ключ
- `yookassa.return-url` — куда вернуть после оплаты (например https://t.me/FlowerDelivery74bot)

Пока тестируем — пополнение через `scripts/deposit_topup.sql`.
